## What I'm Building

### [Griffon](https://github.com/YugmaGandhi/griffon) — OSS Identity Provider
A production-grade, self-hostable authentication platform. No vendor lock-in, no per-user pricing.

- Email/password auth with Argon2id · OAuth2 (Google, GitHub, Microsoft) · JWT RS256
- RBAC · Multi-org support · Webhook subscriptions with HMAC signing
- Audit logging · Prometheus metrics · Brute-force & enumeration protection
- Stack: TypeScript · Node.js · PostgreSQL · Redis · Docker

```bash
npm install @griffon/js
```
